# Quickstart setup

## When to use
Use this to set up the environment before running analysis scripts.

## Expected inputs
- A local clone of the repository
- Python 3.11+ environment

## Expected outputs
- `disRNN_MP` imports without errors
- Editable install ready for notebooks/scripts

## Minimal example

```bash
python -m pip install -e disRNN_MatchingPennies
```

```python
import disRNN_MP
```
